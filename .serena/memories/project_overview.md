# SkillVerse Mobile App Project

## Project Purpose
SkillVerse is a comprehensive learning platform that provides:
- AI-powered career counseling and guidance
- Interactive learning courses and roadmaps
- Smart portfolio building
- Real-time feedback on projects
- Global community connection
- Industry-recognized certifications

The project aims to convert an existing web application (built with Vite + React/TSX) into a Flutter mobile application.

## Current State
- Contains legacy web code in `skillverse-prototype`
- Written in TypeScript with React components
- Features include authentication, courses, dashboard, jobs, portfolio, roadmaps, and AI chatbot
- Uses modern web technologies with responsive design

## Key Features to Convert
1. **Authentication System** - Login, register, email verification, password reset
2. **Dashboard** - User stats, recent courses, learning goals, achievements
3. **Course System** - Browse courses, course details, learning interface
4. **Roadmaps** - Learning paths with progress tracking
5. **AI Chat/Guide** - MeowlChat and MeowlGuide AI assistants
6. **Portfolio** - User profile and skill showcase
7. **Jobs** - Job listings and applications
8. **Help Center** - Support and documentation

## Target Platform
Flutter mobile application targeting both iOS and Android platforms.
